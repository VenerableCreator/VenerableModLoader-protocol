The reason for VenerableModLoader existing is a glaring lack of mods for specific versions, and certain mod loaders doing weird stuff.

VenerableModLoader aims to make mods version agnostic, and easier to make and maintain.

This protocol is here to define how VenerableModLoader works on a high level and what is needed to make it work.

This protocol does not aim to define code optimizations, code structure or generally how it is implemented.
